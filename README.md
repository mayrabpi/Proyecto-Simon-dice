# Simon Dice- Juego de Memoria en Java
Simon Dice es una implementación del clásico juego de memoria "Simon" en java. El juego genera una secuencia aleatoria de colores que el jugador debe memorizar y repetir.
Esta diseñado con dos modos de dificultad y ofrece ayudas para facilitar la jugabilidad.
Características
-Modo Fácil: Secuencia de 4 colores.
-Modo Difícil: Secuencia de 7 colores.
-Ayudas Diponibles: Los jugadores pueden solicitar ayudas para visualizar el siguiente color en la secuencia.
-Puntuación: Los jugadores acumulan puntos por cada secuencia completa correctamente.
